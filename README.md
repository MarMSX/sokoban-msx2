## Sokoban MSX 2 - Solver

Developed by Marcelo Silveira.

Super Sokoban was developed for MSX Dev 2024 contest and can be found here:

[https://www.msxdev.org/2025/01/28/msxdev24-28-super-sokoban/](https://www.msxdev.org/2025/01/28/msxdev24-28-super-sokoban/)

<br>

> Some Dutch MSX veterans joined forces to create a very polished and smooth MSX2 port of the game. It uses the MSX2 capabilities and even features an SCC soundtrack. For those not in the know, in Sokoban, you control a worker in a warehouse, who has to push around boxes to assigned locations. It forces the player to plan ahead, as the space in each storage room is limited.

Got stucked? All the 100 stages of the game were mapped and are now available for solving.
